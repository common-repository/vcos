== Description ==

vcOS - Villa Charcas Online School es un plugin diseñado para proveer una escuela en linea. Se puede crear materias donde cada materia tiene varias lecciones. Cada lección tiene una evaluación y posibilidad de trabajo practico adjunto. Al final de la materia, un examen final. 

== Installation ==

Descarga el plugin de Wordpress.org. Luego de instalar, hay que configurar en opciones->vcOS. 
Despues de crear materias en vcOS-ADMIN, hay que habilitarlas para que sean visibles al usuario.
